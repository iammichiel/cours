What is HTML ?

HTML or HyperText Markup Language is the standard markup language used to create web pages.

History

  • Started as a project at CERN
  • Tim Berners Lee

Html and XML

HTML is an extension of XML

Html can be grouped

  • Structure elements : body, head, html
  • Head elements : base, link, meta, script, style, title
  • Body elements : p, a, div, ul...

Basic elements

  • a : link
  • img : image
  • em, strong : emphasis
