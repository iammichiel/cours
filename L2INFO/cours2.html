Specs of PHP

Syntax


<?php
    // Single line comment
    $number = 10;

    /* Block comment */
    if ($number > 10) {
        echo 'Greater !';
    } else {
        echo 'Lesser or equal';
    }
?>

Comparison

  • < : strictly lesser then
  • > : strictly greater then
  • == / === : equals
  • != / <> / !== : different from
  • && : AND
  • || : OR
