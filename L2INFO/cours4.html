Class


class Person {
    private $name;

    public function setName($name) {
        $this->name = $name;
    }

    public function displayName() {
        echo $this->name;
    }
}

$michiel = new Person();
$michiel->setName('michiel');
$michiel->displayName();

Exercices

Foreach of the following exercices, you must use object oriented design. One file per exercice.

Exercice 1

Create an array with at least 5 people. Each person must have a first name, a last name, a gender and a birthdate.

Display a table with the list of people. For each line, display the full name, the age and gender. Display the number of days before their birthday.

  • Use object design for a Person
  • Use the php object DateTime

Simulating a shopping page
