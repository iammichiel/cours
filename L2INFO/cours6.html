Exercice

User management

Create a table containing `users`. A user is defined by :

  • Username
  • Email
  • Password

Listing the users

Display a simple html table containing all users saved in the database. Add a link to the page to add new users.

Adding a user

Create a webpage with a form that asks the user for all information about the user. If the input is valid, add a new user to the database.

Updating a user

A user can be updated. Display a link in the table, to edit a given user.

Deleting a user

A user can be deleted. Display a delete link for each line in the table.
