TCP/IP

TCP : Transmission control protocol / Internet Protocol

  • Alternative to OSI
  • Model in 4 layers
  • Created by DARPA in 1984
  • Basis of the internet of today

4 layers

  • Application
  • Transport
  • Internet
  • Link

Application layer

  • Application data
  • Examples : HTTP, FTP, SSH...

Transport

  • Error-control
  • Application addressing
  • Port
  • Examples : TCP, UDP

Internet

  • Sends data over differents networks
  • Examples : IP

Link

  • How is the data transmitted ?
  • Token ring, Ethernet

Port

  • For multiple applications on same device
  • Each application has its own port
  • Between 1 and 65535

Reserved ports

  • HTTP : 80
  • SSL : 443
  • FTP : 21
  • SSH : 22
  • SMTP : 25

Octets & bits

  • Bit : basic unit
  • Bit = 0 or 1
  • Octet = 8 bits
  • Octet = Hexadecimal presentation

BIT IS NOT BYTE

What is an IP address?

An Internet Protocol address (IP address) is a numerical label assigned to each device (e.g., computer, printer) participating in a computer network that uses the Internet Protocol for communication

IP address :

  • V4 : 173.194.40.165
  • V4 : 4 octets
  • V6 : 2001:0db8:0000:85a3:0000:0000:ac1f:8001
  • V6 : 16 octets in hexa

TCP : Transmission Control Protocol

TCP provides reliable, ordered and error-checked delivery of a stream of octets

TCP = Connected

  1. Connecting
  2. Transfering
  3. Closing connection

TCP : handshake

  1. A -> B : SYN
  2. B -> A : SYN/ACK
  3. A -> B : ACK
